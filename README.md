# 🕹️ Hangman Game (Python)

This is a text-based Hangman game developed using Python. The player tries to guess a hidden word by entering one letter at a time, with a limited number of attempts.

## 🚀 Features
- Multiple difficulty levels (easy, medium, hard)
- Tracks guessed letters
- Limited attempts with visual hangman stages
- Replay option for continuous gameplay

## 🧠 Concepts Used
- Python loops (while loop)
- Conditional statements (if-else)
- Lists and strings
- Random module
- Functions (modular programming)
